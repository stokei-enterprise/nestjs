## Stokei NestJS core library

##### Description

Stokei NestJS core library.

## Installation

```bash
$ npm install
```

## Running the library

```bash
# development
$ npm run dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
